This is C++ code for a proof-of-concept, Raspberry Pi 5 based bipedal robot.

The code depends on two external, github libraries (beyond the standard Linux libraries):

pipca9685 library by Matthew Barulic: https://github.com/barulicm/PiPCA9685

bn0055_cpp library by Lennart Nachtigall: https://github.com/firesurfer/bno055_cpp
